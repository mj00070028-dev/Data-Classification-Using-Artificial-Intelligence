## Project Overview

This project was developed as part of the DecodeLabs Industrial Training Program 2026 under the Artificial Intelligence track.

The objective of this project is to implement a supervised machine learning classification model using Python. It demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, prediction, and performance evaluation.

The project provides practical experience in applying machine learning techniques to classify data based on learned patterns from a labeled dataset.

---

## Objectives

* Understand the fundamentals of supervised machine learning.
* Load and analyze a structured dataset.
* Perform data preprocessing and exploration.
* Split the dataset into training and testing sets.
* Train a classification model.
* Evaluate model performance using standard metrics.
* Gain practical experience with Python and Scikit-learn.

## Dataset

This project uses a labeled dataset for supervised learning.

Example Dataset: Iris Dataset

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Target Classes

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Machine Learning Algorithm

The classification model is implemented using the Decision Tree Classifier.

Other algorithms that can be used for comparison include:

* Logistic Regression
* K-Nearest Neighbors
* Random Forest Classifier
* Support Vector Machine

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/data-classification-ai.git
```

Navigate to the project directory:

```bash
cd data-classification-ai
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python classification.py
```
